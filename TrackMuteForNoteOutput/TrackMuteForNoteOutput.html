TRACK MUTE FOR NOTE OUTPUT

Bitwig Studio controller script, intended to enable muting the note output of non-audio tracks.

This is achieved by linking the track mute state for instrument tracks (i.e. non-audio tracks) in Bitwig Studio to the activation state of tracks (/ channels).

The (maximum) number of tracks (excluding master track and effect tracks) being observed can be configured by the user.
It is recommended to set this to a relatively large number of tracks, at least as large as the total number of tracks in (all) your projects.
